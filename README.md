# VocabVoyage 词汇之旅

核心想法：让英语学习者在学习的过程中，不单单只是建立英文和中文映射，更能理解每一个单词。我们希望用户能借助大模型生成的内容，亦或是与大模型的交互中学习每个单词背后的内容，比如：词根词源、使用情景、内部逻辑等
